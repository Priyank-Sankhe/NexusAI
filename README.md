# NexusAI 🧠
Your personal AI study operating system.

## Live Demo
[NexusAI](https://nexusai-gcovifrsl5ovpxcse5gun4.streamlit.app/)

## What It Does
Three integrated modes, one shared data layer:
- 💬 Study Chat — AI study assistant powered by Groq/Llama3. No token limits.
- 🎯 GapFinder — Generates DSA problems by topic, evaluates your solution, tracks weak spots.
- ⚡ FlowState — Builds your daily study plan, checks in at 7PM, replans around what actually happened.

## Why I Built It
I'm a career switcher learning software development at Scaler Academy. 
I was losing study time to token limits, passive consumption without retention, 
and days that went off track with no recovery system. I built NexusAI to fix all three.

## Tech Stack
- Python
- Streamlit
- Groq API (Llama 3.1)
- Streamlit Community Cloud
